### Practice Problem 2.3

| Decimal | Binary        | Hex      |
|---------|---------------|----------|
| 0       | 0000 0000     | 0x00     |
| 158     | **1001 1110** | **0x9E** |
| 76      | **0100 1100** | **0x4C** |
| 145     | **1001 0001** | **0x91** |
| **174** | 1010 1110     | **AE**   |
| **60**  | 0011 1100     | **3C**   |
| **241** | 1111 0001     | **F1**   |
| **117** | **0111 0101** | 0x75     |
| **189** | **1011 1101** | 0xBD     |
| **245** | **1111 0101** | 0xF5     |

158 = 9 * 16 + 14  ... E
      0 * 16 + 9   ... 9

76 = 4 * 16 + 12 ... C
     0 * 16 + 4 ... 4

145 = 9 * 16 + 1 ... 1
      0 * 16 + 9 ... 9

AE = 10*16^1 + 14*16^0 = 160+14=174
3C = 3*16^1 + 12*16^0 = 58 + 12 = 60
F1 = 16*15^1 + 1 * 16^0 = 272

0x75 = 7*16^1 + 5 = 117
0xBD = 11*16^1 + 13 = 189
0xF5 = 15 * 16^1 + 5 = 245
